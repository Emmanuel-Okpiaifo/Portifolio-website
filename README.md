## Projects
Here are some of the key projects showcased in my portfolio:

- **Memory Game**: A card matching game built with JavaScript, testing players' memory and speed.
- **Animal Shelter Website (Cloned)**: A clone of an animal shelter website that allows users to view animals for adoption and shelter information.
- **Sudoku Solver**: A logic-based number-placement game built with JavaScript, including an algorithm for solving puzzles.
- **Pig Dice Game**: A simple dice game implementation where players try to reach a target score before losing it all.
- **Cipher**: A JavaScript-based tool that encrypts and decrypts text using simple cipher algorithms.
- **Family Tree**: A JavaScript application to visualize a family tree structure with dynamic node creation.
- **Bank Account Manager**: A simulated banking application that allows users to manage accounts and track transactions.
